## 📌 Описание проекта

Информационная система «StockFlow» предназначена для автоматизации учета материалов на складе предприятия.

**Основные функции:**
- Ведение учета прихода и расхода материалов
- Контроль текущих остатков на складе
- Формирование аналитических отчетов
- Разграничение прав доступа (администратор, модератор, пользователь)
